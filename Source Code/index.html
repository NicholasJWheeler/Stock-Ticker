What Is This Website?

This website was created as the final project for my CSE 383 - D class which focused on web development. My project centers around combining HTML, CSS, JavaScript, Bootstrap, jQuery, and Ajax to make a responsive website that accurately displays stock ticker information using the Polygon.io API.


Features

My stock ticker website has three main pages: the Homepage (where you currently are), Stocks page, and History page. The Stocks page primarily focuses on allowing any user to choose a stock exchange supported by Polygon.io, and find a stock on that exchange that they want to see monetary details or news about. The closing values of the last 7 market days will be shown to the user when the "Details" button is selected. The 5 most recent news articles involving the selected company will be shown when the "News" button is selected, and users can click on the article titles to be brought to the respective website for further reading. All calls made to the API are stored in an SQLite database via phpLiteAdmin, on an Ubuntu Openstack server where this website is hosted. The History page allows a user to input a date and the maximum number of results they would like to see, and results will return from the SQLite database displaying any calls made to the API on the selected date. If a user would like to see the full call results without needing to contact the API, they can click the "Expand" button and the full details about the specified call will be displayed below.
